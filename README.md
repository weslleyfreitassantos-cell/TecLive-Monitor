# 🎥 YouTube Live Monitor V2

Sistema profissional de monitoramento de lives do YouTube para NEOnews.

## 🚀 Início Rápido

```bash
git clone https://github.com/weslleyfreitassantos-cell/youtube-live-monitor-v2.git
cd youtube-live-monitor-v2
npm install
cp .env.example .env
# Edite .env e coloque ADMIN_PASSWORD=sua-senha
node app.js
